**Usage:**

**Command Line:**


## Regenerating the Image

If you need to regenerate or modify Echo's image:

```bash
# Generate default image
python scripts/generate_echo_image.py

# Generate with custom output path
python scripts/generate_echo_image.py path/to/output.png
```

The generation script uses PIL/Pillow to create the image programmatically, ensuring consistency and allowing for customization.

- **Wings for shelter**: Protection and comfort, not escape
- **Always present**: Gentle, constant presence through all states
- **Moon symbol**: Gentle light in darkness, constant presence through cycles
